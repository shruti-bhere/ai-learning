# Matplotlib

## What is Matplotlib?

Matplotlib is a Python library used for creating charts and graphs.

It helps visualize data clearly and is widely used in:

- Data Analysis
- Machine Learning
- Artificial Intelligence
- Data Engineering

## Install

pip install matplotlib

## Import

import matplotlib.pyplot as plt

## Topics Covered

- Line Chart
- Bar Chart
- Pie Chart
- Histogram
- Scatter Plot
- Subplots
- Titles & Labels
- Grid
- Save Figure
