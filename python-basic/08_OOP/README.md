# Object-Oriented Programming (OOP)

## What is OOP?

Object-Oriented Programming (OOP) is a programming paradigm based on classes and objects.

## Why Learn OOP?

- Code Reusability
- Better Project Structure
- Easy Maintenance
- Scalability
- Used in Django, Flask, AI Projects, and Enterprise Applications

## Topics Covered

- Class & Object
- Constructor
- Instance Variables
- Class Variables
- Methods
- Inheritance
- Polymorphism
- Encapsulation
- Abstraction
- Magic Methods

## Learning Outcomes

After completing this section, you will be able to design reusable and maintainable Python applications using OOP concepts.
