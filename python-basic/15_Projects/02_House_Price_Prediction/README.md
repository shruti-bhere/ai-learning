# House Price Prediction

## Project Description

This project predicts the price of a house based on its area using Linear Regression.

## Technologies

- Python
- NumPy
- Pandas
- Matplotlib
- Scikit-learn

## Machine Learning Algorithm

- Linear Regression

## Dataset

| Area | Price |
|------|--------|
|800|40000|
|1000|50000|
|1200|60000|

## Output

The model predicts the price of a house based on the given area.

## Learning Objectives

- Data Loading
- Data Visualization
- Model Training
- Prediction
- Linear Regression
