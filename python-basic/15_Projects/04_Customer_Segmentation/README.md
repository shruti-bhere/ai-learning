# Customer Segmentation

## Project Description

This project groups customers based on their Annual Income and Spending Score using the K-Means Clustering algorithm.

## Technologies

- Python
- NumPy
- Pandas
- Matplotlib
- Scikit-learn

## Machine Learning Algorithm

- K-Means Clustering

## Features Used

- Annual Income
- Spending Score

## Concepts Learned

- Unsupervised Learning
- Clustering
- Data Visualization
- Customer Analysis

## Output

The program assigns every customer to a cluster and displays the cluster centroids with a scatter plot.
