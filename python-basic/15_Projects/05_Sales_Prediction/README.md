# Sales Prediction using Machine Learning

## Project Description

This project predicts product sales based on advertising budgets spent on TV, Radio, and Newspaper using Linear Regression.

- NumPy
- Pandas
- Scikit-learn

## Machine Learning Algorithm

- Model Training
- Prediction
- Model Evaluation
