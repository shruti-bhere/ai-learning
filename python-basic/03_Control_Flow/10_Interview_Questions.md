# Interview Questions

## 1. What is a conditional statement?

A conditional statement executes different blocks of code depending on whether a condition is True or False.

---

## 2. Difference between if and if-else?

- if executes a block only when the condition is True.
- if-else provides an alternative block when the condition is False.

---

## 3. What is nested if?

A nested if is an if statement inside another if statement.

---

## 4. What is the ternary operator?

It is a short form of if-else.

Example:

result = "Pass" if marks >= 40 else "Fail"

---

## 5. What is match-case?

match-case is Python's pattern matching statement introduced in Python 3.10. It is similar to the switch statement in other programming languages.
