# Interview Questions

## 1. What are operators?

Operators are symbols that perform operations on operands.

---

## 2. Name different types of operators in Python.

- Arithmetic
- Assignment
- Comparison
- Logical
- Identity
- Membership
- Bitwise

---

## 3. Difference between == and is?

== compares values.

is compares whether two variables refer to the same object in memory.

---

## 4. Difference between / and //?

/ returns floating-point division.

// returns floor (integer) division.

---

## 5. What is operator precedence?

Operator precedence determines the order in which operators are evaluated in an expression.
