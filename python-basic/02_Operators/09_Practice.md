# Practice Questions

## Basic

1. Add two numbers.
2. Find the remainder when dividing two numbers.
3. Calculate the square of a number.
4. Compare two numbers.
5. Check whether a number is greater than 100.

## Intermediate

6. Check if a person is eligible to vote using logical operators.
7. Check if a character exists in a string.
8. Check whether a number is even or odd.
9. Demonstrate assignment operators.
10. Use operator precedence in a mathematical expression.

## Challenge

Create a simple calculator using arithmetic operators.
