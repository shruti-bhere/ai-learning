# Interview Questions

## 1. What is a function?

A function is a reusable block of code that performs a specific task.

---

## 2. Why use functions?

- Code Reusability
- Better Readability
- Easy Maintenance
- Modularity

---

## 3. Difference between parameters and arguments?

Parameters are variables in the function definition.

Arguments are the values passed to the function when calling it.

---

## 4. What is a return statement?

The return statement sends a value back to the caller.

---

## 5. What is recursion?

Recursion is a technique where a function calls itself until a base condition is met.

---

## 6. What is a lambda function?

A lambda function is a small anonymous function written in one line.
