# Python Modules and Packages

## What is a Module?

A module is a Python file (.py) that contains functions, classes, and variables.

Example:

math.py

student.py

calculator.py

## What is a Package?

A package is a folder containing multiple Python modules.

## Why Use Modules?

- Code Reusability
- Better Organization
- Easy Maintenance
- Large Project Structure
