# Interview Questions

## 1. What is Python?

Python is a high-level, interpreted programming language known for its simple syntax and readability.

---

## 2. What is a variable?

A variable is a named location in memory used to store data.

---

## 3. Name Python data types.

- int
- float
- str
- bool
- list
- tuple
- set
- dictionary

---

## 4. What does input() do?

It reads input from the user and returns it as a string.

---

## 5. What is type casting?

Type casting is converting one data type into another using functions such as int(), float(), or str().
