# Pandas

## What is Pandas?

Pandas is a Python library used for data manipulation and analysis.

It provides two main data structures:

- Series (1D)
- DataFrame (2D)

## Why Learn Pandas?

- Read CSV/Excel files
- Clean data
- Analyze datasets
- Used in Machine Learning
- Used in Data Engineering
- Used in AI Projects

## Install

pip install pandas

## Import

import pandas as pd
